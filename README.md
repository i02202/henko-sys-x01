# Henko Sys x01

> Sovereign AI Operations Platform — A modular, self-hosted, self-improving multi-agent system.

## What is Henko?

Henko (Japanese for "transformation") is an autonomous AI platform that researches, trades, and builds software — running entirely on your own infrastructure.

## Modules

- **ALPHA** — AI-powered trading & finance (Hyperliquid, backtesting, strategy evolution)
- **FORGE** — Autonomous software development (BMAD methodology, Claude Code Teams, design-to-code)
- **INTEL** — Research & intelligence (geopolitical monitoring, tech intel, RAG knowledge base)
